# Indice de Diagramas

Esta carpeta almacena diagramas que mejoran el entendimiento del sistema de reportes asincronos de Prosperas.

Conjuntos de diagramas actuales:
- architecture/system-architecture.md
- workflow/job-lifecycle.md
- workflow/ci-cd-overview.md
- user-flow/report-request-journey.md

Reglas para diagramas:
- preferir Mermaid en Markdown para facilitar versionado
- mantener los diagramas alineados con el SSOT activo
- actualizar diagramas cuando cambien de forma relevante la arquitectura, el despliegue o el flujo de usuario
